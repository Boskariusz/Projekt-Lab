Projekt buzzera oparty o płytkę Arduino Uno

Opis projektu
Polega na stworzeniu kodu na arduino, który pozwoli grać melodie za pomocą buzzera. Moim przykładową melodia jest dzwonek 
firmy nokia.

Potrzebne elementy:
- płytka arduino uno
- buzzer
